  • 作品２「気温の変化に合わせた服装を提案する機能」(Javascript)
    Fashion Advisor

    works2 : Websiteを見る

    works2 : ソースコードを見る

    ▲自主制作作品２のwebサイトとソースコードはこちらからご覧ください。 ソースコードは Githubにてご覧いただけます。
    制作概要を見る

    制作内容

    作品２「Fashion Advisor」

    使用言語

    HTML / CSS / Javascript

    この制作で工夫したこと

    ダイスで用いたコードを応用し、 AIの生成コードも参考に必要な機能を作っています。
    私自身、日々の気温の変化で着る服を迷うことが多くあったため、 気温によって服装を提案してくれる機能を作成しました。
    寒いときや熱いときの背景をCSSとJSの組み合わせで変更させたり、 温度計に変化を加えたりすることで、気温の変化を視覚的に感じ取れるようにしています。

    この制作で苦労したこと

    どのあたりの温度にどの服装を設定するのが適正なのかの確認や、温度ごとの条件設定、
    温度計のシームレスな動きと色の変化の実装
    他、マウスで服の絵を描いたことなどで苦労しました。
