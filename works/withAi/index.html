    制作内容

    AI活用作品・1「簡易版入退室管理システム」
    ・常に現在時刻が画面上に表示されます。
    ・名前をJSONデータで取り込んでおき、セレクトボックスで入室者を選択。
    ・すでに入室しているときはエラー表示が出ます。
    ・入室の時は、退室予定時刻をセレクトボックスで選択し入室登録。
    ・入室中はその人物の在室状況がリストに表示され、退室予定時刻、予定時刻までのカウントダウンも隣に表示されます。
    ・予定退室時刻が迫ると、対象となる人のデータ表示色が変化します。
    ・退室の時は入室中の名前表示をクリックすると退室処理ができます。 ・画面下には入退室の記録が表示されます。
    ・履歴をダウンロードすることができます。
    ・リセットのボタンを押すと、すべての履歴が消去されます。
    ・リセットボタンを押すか、ブラウザデータが消えるまで履歴は保持されます。

    使用言語

    HTML / CSS / Javascript

    この制作で工夫したこと

    前職の学童クラブで勤めていた頃、児童の入退室システムに興味があり、 プログラミングを学ぶ事で機能を理解、作成できるようになりたいと考えていました。
    機能をそろえるには難易度が高く、私一人では実現が困難でしたが、AIを活用することで実現できました。 学習のため、なるべく私が理解できる範囲に収める形で、HTML/CSS/JSで言語でできる簡易的な入退室システムを作成しています。 ＊オフラインでもブラウザを使用できれば機能します。

    制作内容

    AI活用作品・２「〇×クイズジェネレーター」
    ・出題者が事前にクイズと答えを一行一問ずつ入力
    ・開始ボタンで制限時間付きでクイズセットが開始されます。
    ・クイズの正当数をカウント。
    ・作ったクイズはブラウザに保存すれば、ブラウザのデータが消えるまではそのまま利用可能。
    ・作ったクイズのtxt形式データをダウンロードして読みだせば、再利用可能。

    使用言語

    HTML / CSS / Javascript

    この制作で工夫したこと

    学童クラブでよく行われるクイズイベントで使える〇×クイズジェネレーターを作成してみました。
    難易度が高く、私一人では実現が困難でしたが、AIを活用し、なるべく私が理解できる範囲に収める形で、 HTML/CSS/JSで言語でできる簡易的な入退室システムを作成してみました。 ＊オフラインでもブラウザを使用できれば機能します。

フロントエンド: HTML, CSS, JavaScript
