  • 課題３「ドーナツショップサイト」は近日中にUPいたします。

    Exercise3 : Websiteを見る

    Exercise3 : ソースコードを見る

    ▲訓練校の制作課題３のwebサイトとソースコードはこちらからご覧ください。 ソースコードは Githubにてご覧いただけます。
    制作概要を見る

    制作内容

    課題３「ドーナツショップサイト」
    制作時間：78時間

    使用言語

    HTML / CSS / JavaScript /SQL/PHP

    制作前の主な学習内容

    PHP / SQL / サーバー上でのDATABASE作成（XAMPP利用）活用基礎
    学習時間：78時間
    期間：1月5日〜1月22日

    この制作で工夫したこと

    初めてのDB、SQLとPHPを用いたWEBサイト制作でした。HTML・CSSでベースのページを作成後、 PHP化し、サーバー上で機能する処理などの実装を行っています。 演習期限内では製作期間内での課題要件をクリアすることに焦点を絞り、 第一段階として、ページへの商品カート機能、会員登録機能、ログイン機能を実装しました。 その後、購入手続きなど他の機能については追加で作成予定です。

    この制作で苦労したこと

    PHPでの初めての作成で、まずHTMLやCSSとの表示部分の役割分け、画像の命名上の注意などの理解から苦労しました。 また、HTMLからPHPへの変更で、コードの記述間違いによりDBからの取得エラー、 処理の対象の指定間違いによるエラーなどにより、思うように処理が働かない事や、 HTMLだけの段階では表示されていたものが正しく表示されないなど、PHPならではの難しさを実感しました。
    この時点の自身のPHPへの理解度と制作物の量に対し、時間に限りがあったため、画面への処理状況の表示機能や、 AIなどによるコードの実行性の可否判断を補助的に活用することで期限内に一通りの機能を実装しています。
